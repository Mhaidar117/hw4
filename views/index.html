What is Machine Learning

"Machine Learning is the field of study that gives computers the ability to learn without being explicitly programmed."

- Arthur Samuel, 1959

We will be explaining a few core concepts; supervised and unsupervised learning, as well as online and batch learning. These concepts are used to "train" your machine or system.

The system is typically given preliminary data to create an algorithm, this data is usually refered to as a "training set".
